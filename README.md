# Instructions

There are a few required steps for this proyect to run:

1. Copy `.env.sample` to `.env` in the root directory and complete `REACT_APP_MAPBOX_API_TOKEN` with a valid token from [Mapbox](https://www.mapbox.com/).
2. Copy the data file provided `frontend-geo.json` into `src/data/frontend-geo.json`.
3. Run `npm start`.

You can see more details below.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
